
# Cursor Clone Website

## Sections Made

1. **Top Menu** - Logo, links, Sign In and Download buttons
2. **Big Welcome Section** - Headline "Build Fast" with buttons
3. **Code Windows** - Code examples shown in boxes
4. **Company Logos** - Pictures of companies using Cursor
5. **Features** - Three main features with AI and code
6. **Section Title** - "The new way to build software"
7. **Feature Cards** - Six cards showing different features
8. **Image Area** - Big picture with text about Cursor
9. **Try Cursor Button** - Final call to download
10. **Footer** - Links and info at the bottom

## Fonts Used

**Main Font** - System fonts like Segoe UI, Arial (clean and modern)

**Code Font** - Courier New (for code examples)

**Font Sizes**:
- Big titles: Large (for headlines)
- Regular text: Medium (easy to read)
- Small text: Small (for navigation and footer)

## Colors Used

**Dark Background** - Black (#000000)

**Text** - White (#ffffff)

**Accents**:
- Blue (#4a9eff) - For special parts
- Gray (#999999) - For less important text

**Code Colors**:
- Purple - Keywords
- Yellow - Function names
- Orange - Text in quotes
- Green - Comments
